# CS545-Tokyo-Drifters-Final-Project

## Description
Web Application Architecture course project. Our application is built for car rental services. They register all cars in the system which are rented to customers. In other words, when a customer needs a car in city, they just came to the car rental company and the service worker checks out the car for customer. After using the car, customer returns it as the way he or she got it from a service worker.

## Technologies
Java Spring-MVC, JSP, HTML, CSS, JSP, Ajax, JavaScript.
